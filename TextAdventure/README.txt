Text Adventure README

To run the executable:
       On a Windows machine, double-click to run the executable.
How to play:
- Use keyboard input
- The available inputs are always listed at the end of the displayed text
- Type the exact text and hit enter to use one of the available commands
- The game is over when your health reaches zero or when you defeat all enemies
- Slicing is a horizontal attack, while stabbing is a forward attack
- Base your attacks and dodges on the position of the enemy to succeed
